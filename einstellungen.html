E-Mail-Benachrichtigungen

ℹ️ Hinweis: Die Server-Verbindung (IMAP/SMTP) wird über das Backend verwaltet. E-Mail-Benachrichtigungen werden an die E-Mail-Adresse des jeweiligen Kunden gesendet.
Hinweis: Die E-Mail-Funktionalität erfordert eine Backend-Integration. Derzeit werden die Benachrichtigungen nur in der Warteschlange gespeichert.

Verhalten der Benachrichtigungen:
  • Überfällige Rechnungen: Werden automatisch zur Mail-Warteschlange hinzugefügt, wenn Sie das Dashboard besuchen. Sie können diese dann zur Bestätigung prüfen.
  • Alle anderen Benachrichtigungen: Sie werden beim Erstellen, Ändern oder Löschen gefragt, ob Sie eine E-Mail-Benachrichtigung senden möchten.

Automatische Bank-E-Mail Prüfung (IMAP)

Wenn aktiviert, prüft das System beim Öffnen des Dashboards automatisch Ihren E-Mail-Posteingang auf Zahlungsbenachrichtigungen Ihrer Bank und markiert erkannte Rechnungen ohne Ihr Zutun als bezahlt.

Die Zugangsdaten (E-Mail-Adresse und Passwort) werden aus der bestehenden Backend-Konfiguration (backend/config.json) übernommen. Hier konfigurieren Sie nur den IMAP-Server Ihres Postfachs.

Typische IMAP-Einstellungen:
  • World4You: imap.world4you.com · Port 993 · SSL
  • Gmail: imap.gmail.com · Port 993 · SSL (App-Passwort erforderlich)
  • Outlook: outlook.office365.com · Port 993 · SSL
Ordner, in dem die Bank-E-Mails eingehen. Unterordner werden mit / getrennt angegeben (z. B. INBOX/finanzen).

Administrator-Zugangsdaten ändern

Hier können Sie die E-Mail-Adresse oder das Passwort des Administrator-Kontos ändern. Sie können entweder nur die E-Mail-Adresse, nur das Passwort oder beides auf einmal ändern. Zur Sicherheit ist die Eingabe Ihres aktuellen Passworts erforderlich.
